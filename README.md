### 📂 Featured Projects

#### 🚀 **[BashaSethuu](https://github.com/pradeepkambalapally/bashasethuu)**
*Banjara-Telugu-English Speech and Text Translation Platform*
- Decoupled **MERN client-server** architecture using Vite React and Node/Express.
- Integrated browser **Web Speech API** for real-time Speech-to-Text.
- Developed a custom **phonetic consonant-matching algorithm** in the backend service layer for robust dialect lookup.
- Implemented streaming **Text-to-Speech proxy** for Telugu audio clips to bypass CORS limitations.
- Secured using stateless **JWT authentication** with route protection middleware.
- 🔗 **Live Demo:** [https://bashasethu-b28v.onrender.com/](https://bashasethu-b28v.onrender.com/)
